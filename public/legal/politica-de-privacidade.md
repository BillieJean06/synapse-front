# Política de Privacidade

Esta política descreve como a plataforma Neodits coleta e protege os dados.

## Dados coletados

- Nome
- E-mail
- Nome de usuário
- Dados de autenticação

## Uso dos dados

Os dados são usados apenas para funcionamento da plataforma.

## Segurança

Utilizamos criptografia e autenticação JWT.

## Direitos do usuário

O usuário pode solicitar exclusão de dados a qualquer momento.
